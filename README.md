# suspicious-input-scanner
Developed a Python-based security tool that detects malicious input patterns associated with SQL Injection, Cross-Site Scripting (XSS), and Command Injection attacks.

Implemented a rule-based detection engine with risk scoring (Low, Medium, High) and security event logging to simulate basic SOC-style monitoring.

This project helped reinforce understanding of OWASP Top 10 vulnerabilities and foundational detection engineering concepts.
